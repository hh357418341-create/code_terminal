# Handoff Template

Use this when pausing, resuming, compacting context, or handing work to another agent.

```text
Goal:
- <current user goal>

Current state:
- <branch, commit, dirty files, running processes>

Changed files:
- <file>: <why>

Commands run:
- <command>: <result summary>

Validation:
- <checks passed>
- <checks failed or not run, with reason>

Deployment:
- <target, method, service, URL, verification>

Secrets:
- No passwords/tokens were written to files or commits.
- Any required token/password must be supplied interactively or read from existing server env.

Next steps:
- <ordered list>
```
